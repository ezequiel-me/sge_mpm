## Ejercicio 3
```python

numeros = [1, 2, 3, 4, 5]
numerosSuma = reduce(lambda acc, numero: acc+numero, numeros, 0)
```

## Ejercicio 5

```python
palabras = ["sol", "luna", "estrella", "cielo", "mar"]

    diccionario.update({cantidadLetras : palabrasEncontradas})
```
